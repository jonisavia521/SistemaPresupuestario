Actualización de Padrón ARBA

Actualiza masivamente los porcentajes de percepción ARBA de todos los clientes en una sola operación.

Acceso

Ruta: Ventas > Actualizar Padrón ARBA

¿Cuándo Actualizar?

Cuando ARBA publica nuevos porcentajes, típicamente:

  • Anualmente (generalmente en enero)
  • Por cambios legislativos específicos
  • Cuando se modifican actividades

Qué Hace Esta Operación

La actualización:

  • Busca TODOS los clientes con Tipo IVA = "Responsable Inscripto"
  • Asigna nuevos porcentajes ARBA según provincia/actividad
  • Los clientes con otro Tipo IVA NO se modifican
  • Se guardan automáticamente en la base de datos

Procedimiento Paso a Paso

Paso 1: Abrir Ventana

  1. Ve a Ventas > Actualizar Padrón ARBA
  2. Se abrira una ventana con instrucciones

Paso 2: Revisar Información

  1. Lee cuidadosamente la advertencia
  2. Verifica que tengas los porcentajes correctos
  3. Consulta con ARBA si es necesario

Paso 3: Confirmar Actualización

  1. Haz clic en "Actualizar Padrón"
  2. Sistema pedirá confirmación (¡IMPORTANTE!)
  3. Confirma que deseas actualizar TODOS los clientes

Paso 4: Esperar a que Finalice

  1. Verás una barra de progreso
  2. NO CIERRES LA APLICACIÓN durante el proceso
  3. Cuando termine, veras mensajes de éxito

Advertencias Importantes

?? ANTES DE ACTUALIZAR:

  • Verifica que tengas los porcentajes CORRECTOS
  • Haz un BACKUP de tu base de datos
  • No interrumpas el proceso (puede corromper datos)
  • Usa solo si ARBA publica nuevos porcentajes

Después de Actualizar

Una vez completada la actualización:

  • Todos los clientes Responsables Inscriptos tienen nuevos porcentajes
  • Nuevos presupuestos usarán los porcentajes actualizados
  • Presupuestos anteriores mantienen sus porcentajes original
  • Verifica algunos clientes para confirmar la actualización

Si Algo Sale Mal

Si viste error durante la actualización:

  • La operación se cancela automáticamente
  • La base de datos vuelve al estado anterior
  • Puedes reintentarlo

Si necesitas revertir cambios:

  • Restaura el backup que hiciste antes
  • O contacta a soporte técnico

Verificación Manual

Después de actualizar, verifica algunos clientes:

  1. Ve a Maestros > Clientes
  2. Edita un cliente importante
  3. Revisa el campo "Alícuota ARBA"
  4. Confirma que tenga el nuevo porcentaje
