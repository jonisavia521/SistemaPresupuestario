Gestión de Usuarios y Seguridad

Solo el administrador puede crear y gestionar usuarios. Los usuarios tienen acceso según su rol (Vendedor, Aprobador, Facturador, etc.).

Acceso

Ruta: Maestros > Usuarios (solo para administrador)

Crear Usuario

  1. Haz clic en "Nuevo"
  2. Ingresa nombre (completo del usuario)
  3. Ingresa usuario (para login)
  4. Ingresa contraseña temporal
  5. Asigna roles/permisos (Vendedor, Aprobador, etc.)
  6. Guarda

Seguridad

  • Cada usuario tiene su propio login y contraseña
  • Las contraseñas deben cambiarse en el primer acceso
  • Se recomienda cambiar cada 90 días
  • Usa contraseñas fuertes (mayúsculas, números, símbolos)
  • No compartas contraseñas

Roles y Permisos

Típicamente existen estos roles:

  • Administrador: Acceso total al sistema
  • Vendedor: Crear/editar presupuestos
  • Aprobador: Revisar y aprobar presupuestos
  • Facturador: Emitir facturas
  • Ver solo: Consultar sin modificar

Cambiar Contraseña de Usuario

  1. Si eres el usuario: Generalmente en un menú de perfil
  2. Si eres administrador: Edita el usuario y resetea contraseña
  3. El usuario debe cambiarla en el próximo login

Eliminar Usuario

Solo el administrador puede eliminar usuarios.

Generalmente, es mejor desactivar que eliminar, para mantener historial.
