Gestión de Listas de Precios

Cree y administre listas de precios diferenciadas para distintos clientes o segmentos del mercado.

Concepto de Listas de Precios

Lista de Precios: Conjunto de precios aplicables a productos específicos, generalmente vinculados a un cliente o grupo de clientes. Permite ofrecer precios diferenciados según acuerdos comerciales.

Acceso al Módulo de Listas de Precios

Ruta: Menú Principal -> Maestros -> Listas de Precios

Este módulo tiene dos secciones: Listas (encabezados) y Detalle (precios específicos).

Crear una Nueva Lista de Precios

Paso 1: Crear el Encabezado

  1. Haga clic en el botón Nuevo
  2. Complete los datos de la lista:
    • Nombre: Descripción clara (ej: "Cliente VIP", "Mayoristas")
    • Descripción: Información adicional
    • Activa: Marque para usar en presupuestos
  3. Haga clic en Guardar

Datos de la Lista

Campo Descripción Requerido
Código Identificador único (asignado automáticamente) Sí
Nombre Nombre descriptivo de la lista Sí
Descripción Notas adicionales No
Activa Disponible para usar en presupuestos Sí
Vigencia Desde Fecha de inicio de validez No
Vigencia Hasta Fecha de vencimiento No

Agregar Precios a la Lista

Paso 2: Cargar Detalles de Precios

  1. Seleccione la lista creada
  2. En la sección de Detalles, haga clic en Nuevo
  3. Complete la información:
    • Producto: Seleccione de la lista
    • Precio: Ingrese el precio para este cliente
    • Descuento %: Descuento adicional (opcional)
  4. Haga clic en Guardar
  5. Repita para cada producto en la lista

Datos del Detalle

Campo Descripción Requerido Nota
Producto Producto al que aplica este precio Sí Seleccionar de lista desplegable
Precio Precio unitario para este cliente Sí Sin IVA
Descuento % Descuento adicional en porcentaje No Ej: 5, 10, 15

Ejemplos de Listas de Precios

Casos de Uso Comunes:

  • Cliente VIP: Precios reducidos para clientes principales
  • Mayoristas: Precios especiales por volumen
  • Distribuidores: Precios mayoristas con descuento
  • Promoción Temporal: Precios especiales por período
  • Segmento Educativo: Precios especiales para instituciones
  • Mercados Internacionales: Precios en otra moneda

Asociar Lista de Precios a Cliente

Para que un cliente use automáticamente una lista específica:

  1. Abra el módulo de Clientes
  2. Edite el cliente deseado
  3. En el campo "Lista de Precios", seleccione la lista
  4. Haga clic en Guardar

Ahora, al crear un presupuesto para este cliente, el sistema usará automáticamente esta lista.

Uso en Presupuestos

Cuando crea un presupuesto:

  1. Selecciona un cliente
  2. Si tiene lista asignada, los precios se cargan automáticamente
  3. Si no tiene lista, usa los precios base de productos
  4. Puede cambiar precios manualmente en el presupuesto

Editar Lista de Precios

Modificar Encabezado:

  1. Seleccione la lista
  2. Haga clic en Modificar
  3. Actualice los datos
  4. Haga clic en Guardar

Modificar Detalle (Precio):

  1. Seleccione la lista
  2. En los detalles, seleccione la línea a modificar
  3. Haga clic en Modificar
  4. Actualice el precio
  5. Haga clic en Guardar

Eliminación de Precios

Para eliminar un producto de una lista de precios:

  1. Seleccione la lista
  2. En los detalles, seleccione la línea
  3. Haga clic en Eliminar
  4. Confirme la eliminación

Solo se elimina de la lista, el producto sigue existiendo.

Gestión de Vigencia

Puede establecer fechas de validez:

  • Vigencia Desde: A partir de cuándo es válida
  • Vigencia Hasta: Hasta cuándo es válida

Permiten crear listas temporales para promociones o cambios de precio futuros.

Copiar Lista de Precios

Para crear una lista basada en otra existente:

  1. Seleccione la lista original
  2. Haga clic en Copiar (si está disponible)
  3. Ingrese un nombre para la nueva lista
  4. Se crearán todos los detalles automáticamente
  5. Ajuste los precios necesarios

Reportes de Listas de Precios

Análisis disponibles:

  • Comparativa de Precios: Entre listas y precios base
  • Variación de Precios: Cambios en el tiempo
  • Clientes por Lista: Qué clientes usan cada lista
  • Vigencia: Listas activas y vencidas

Mejores Prácticas

  • Nombres descriptivos: Facilita identificación
  • Precios consistentes: Revise antes de activar
  • Vigencia clara: Indique período de validez
  • Documentación: Registre razón de cambios
  • Revisión periódica: Actualice según inflación
  • Control de cambios: Mantenga versiones antiguas
  • Comunicación: Informe cambios a equipo de ventas

Flujo de Precios

¿Qué precio se usa en un presupuesto?

  1. Primero: Si existe precio en lista asignada al cliente -> Usa ese
  2. Segundo: Si no existe precio en lista -> Usa precio base del producto
  3. Tercero: Puede editar manualmente en el presupuesto
Consejo: Cree una lista "Precio Base" con los precios estándar como referencia.
