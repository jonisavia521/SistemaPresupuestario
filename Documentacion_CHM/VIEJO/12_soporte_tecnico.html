Soporte Técnico y Troubleshooting

Solucione problemas comunes y contacte al equipo de soporte cuando sea necesario.

Problemas de Conexión

No puedo conectarme al sistema
Soluciones:
  1. Verifique que está utilizando el usuario y contraseña correctos (sensibles a mayúsculas)
  2. Confirme que la cuenta no está bloqueada o desactivada
  3. Verifique su conexión a Internet/red
  4. Intente desconectar y reconectar a la red
  5. Reinicie la aplicación
  6. Si persiste, contacte al administrador (la base de datos podría no estar disponible)
"Error de conexión a base de datos"
Soluciones:
  1. Verifique su conexión a Internet (la base de datos está en servidor remoto)
  2. Asegúrese que el servidor SQL Server está encendido y disponible
  3. Espere unos minutos y reintente (podría ser un problema temporal)
  4. Reinicie la aplicación
  5. Si está en red corporativa, verifique con IT que el puerto 1433 está abierto
  6. Contacte al administrador técnico
La aplicación es muy lenta
Soluciones:
  1. Cierre programas innecesarios para liberar memoria
  2. Verifique su velocidad de conexión a Internet
  3. Intente conectarse en un horario de menor uso
  4. Reinicie la aplicación y el navegador
  5. Si está trabajando con muchos datos, intente aplicar filtros para reducir volumen
  6. El problema podría ser del servidor (contacte al administrador)

Problemas de Autenticación

Olvidé mi contraseña
Soluciones:
  1. Contacte al administrador del sistema
  2. Proporcione su nombre de usuario y nombre completo
  3. El administrador generará una contraseña temporal
  4. Cámbiela en el primer acceso
  5. No comparta su contraseña - el administrador nunca la pedirá
Mi cuenta está bloqueada
Soluciones:
  1. Podría bloquearse por múltiples intentos de login fallidos
  2. Espere 30 minutos y reintente
  3. Si persiste, contacte al administrador para desbloquear

Problemas de Permisos

No veo algunos módulos en el menú
Soluciones:
  1. Es normal si no tiene permisos asignados para esos módulos
  2. Contacte al administrador para solicitar acceso
  3. El administrador puede asignarle los permisos necesarios
"Acceso denegado" al abrir un módulo
Soluciones:
  1. No tiene permiso para ese módulo
  2. Contacte al administrador para solicitar acceso
  3. Espere a que le asigne el permiso y reinicie la aplicación

Problemas de Presupuestos

No aparece el cliente que busco
Soluciones:
  1. El cliente podría estar marcado como "Inactivo"
  2. Desactive el filtro de clientes activos
  3. Verifique que el cliente está creado en Maestros -> Clientes
  4. Use búsqueda parcial (busque por parte del nombre)
El precio del producto no es el correcto
Soluciones:
  1. Verifique qué lista de precios está asignada al cliente
  2. Si no hay lista, usa el precio base del producto
  3. Puede editar manualmente el precio en el presupuesto
  4. Si cree que el precio está mal, contacte al administrador
No puedo editar un presupuesto
Soluciones:
  1. Solo se pueden editar presupuestos en estado "Emitido" o "Aprobado"
  2. Los presupuestos "Facturados" o "Rechazados" no se pueden editar
  3. Para cambiar uno facturado, debe copiar y crear uno nuevo

Problemas de Facturación

Error al emitir factura
Soluciones:
  1. Verifique que el presupuesto está en estado "Aprobado"
  2. Confirme que los datos fiscales son correctos (CUIT, razón social)
  3. Revise la configuración de empresa en Configuración General
  4. Asegúrese que el punto de venta está configurado
  5. Intente desconectar/reconectar y reintente
  6. Si persiste, contacte al soporte técnico
"Error al conectar con AFIP"
Soluciones:
  1. Verifique su conexión a Internet
  2. AFIP podría estar en mantenimiento - intente más tarde
  3. Verifique el estado de AFIP en www.afip.gov.ar
  4. Intente nuevamente en otro horario
  5. Si es un problema persistente, contacte al soporte
"Datos de factura inválidos"
Soluciones:
  1. Verifique que el CUIT es válido (formato: XX-XXXXXXXX-X)
  2. Confirme que la razón social es exacta
  3. Revise que el tipo de factura (A, B, C) coincide con el cliente
  4. Asegúrese que todos los campos requeridos están completos
  5. Contacte al administrador si el error persiste
No puedo descargar el PDF
Soluciones:
  1. Verifique que tiene espacio en disco
  2. Intente descargar en otra carpeta
  3. Verifique que su navegador permite descargas
  4. Cierre el PDF anterior si lo tiene abierto
  5. Reinicie la aplicación e intente nuevamente

Problemas Técnicos Generales

La aplicación se congela o no responde
Soluciones:
  1. Espere un momento (podría estar procesando)
  2. Si no responde, presione Alt+F4 para cerrar
  3. Vuelva a abrir la aplicación
  4. Si el problema persiste, reinicie la computadora
  5. Contacte al soporte técnico si continúa
Mensaje de error incomprensible
Soluciones:
  1. Anote el código de error exacto
  2. Tome una captura de pantalla del mensaje
  3. Contacte al soporte técnico con esta información
  4. Describa qué estaba haciendo cuando ocurrió

Pérdida de Datos

Perdí datos que estaba completando
Soluciones:
  1. Si no hizo clic en "Guardar", los datos se pierden
  2. SIEMPRE haga clic en "Guardar" antes de cerrar
  3. No hay recuperación de datos no guardados
  4. Para el futuro, guarde frecuentemente mientras trabaja
Se perdieron datos de la base de datos
Soluciones:
  1. Contacte inmediatamente al administrador
  2. No realice más cambios en el sistema
  3. El administrador intentará recuperar de un respaldo
  4. En el futuro, asegúrese que se hacen respaldos regularmente

Información para Reportar un Problema

Cuando contacte al soporte, proporcione:

  1. Descripción clara: ¿Qué está tratando de hacer?
  2. Pasos reproducibles: Cómo puede reproducir el problema
  3. Mensaje de error: Código y texto exacto del error
  4. Captura de pantalla: Si es posible, adjunte imagen
  5. Su usuario: Nombre de usuario con el que accede
  6. Cuándo ocurre: ¿Siempre o a veces?
  7. Ambiente: ¿Está conectado a red? ¿VPN?
  8. Versión: ¿Cuál es la versión del sistema que usa?

Mejores Prácticas para Evitar Problemas

  • Guarde frecuentemente: No pierda datos por olvido
  • Verifique datos antes de guardar: Revise completitud
  • No cierre sin guardar: Siempre guarde cambios
  • Actualice contraseña regularmente: Cada 90 días
  • Reporte problemas de inmediato: No espere
  • Contacte con administrador: Para cambios de configuración
  • No modifique configuración: A menos que sea autorizado
  • Mantenga antivirus actualizado: En su PC
