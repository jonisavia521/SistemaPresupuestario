Gestión de Clientes

Administre el registro de clientes y mantenga actualizada su información de contacto y comercial.

Acceso al Módulo de Clientes

Ruta: Menú Principal -> Maestros -> Clientes

Este módulo permite crear, consultar, modificar y eliminar clientes de su base de datos.

Vista de Listado de Clientes

En la pantalla principal verá una tabla con todos los clientes registrados:

Funcionalidades:

  • Búsqueda: Busque por código, razón social o CUIT
  • Orden: Haga clic en los encabezados para ordenar
  • Nuevo: Crear un cliente nuevo
  • Modificar: Editar cliente seleccionado
  • Eliminar: Borrar cliente (solo si no tiene presupuestos)
  • Ver: Consultar detalles sin modificar

Crear un Nuevo Cliente

  1. Haga clic en el botón Nuevo
  2. En el formulario, ingrese los datos del cliente
  3. Haga clic en Guardar
  4. El sistema le asignará automáticamente un código

Datos del Cliente

Campo Descripción Requerido Formato
Código Identificador único (asignado automáticamente) Sí Numérico
Razón Social Nombre completo del cliente o empresa Sí Texto (máx 100 caracteres)
CUIT Código de Impuesto Único Tributario No XX-XXXXXXXX-X
Dirección Domicilio comercial No Texto
Localidad Ciudad donde reside el cliente No Texto
Provincia Estado o provincia No Seleccionar de lista
Teléfono Número de contacto principal No Texto
Email Correo electrónico para comunicaciones No email@example.com
Condición IVA Situación fiscal del cliente Sí Responsable, Monotributista, etc.
Condición de Pago Forma de pago habitual No Contado, Cuenta Corriente, etc.

Editar Cliente Existente

  1. En la lista, seleccione el cliente a modificar
  2. Haga clic en el botón Modificar
  3. Actualice los campos necesarios
  4. Haga clic en Guardar

Buscar Cliente

El sistema proporciona búsqueda en tiempo real:

  • Por Código: Busque por el número asignado
  • Por Razón Social: Nombre del cliente (búsqueda parcial disponible)
  • Por CUIT: Número de impuesto

Escriba en el campo de búsqueda y el sistema filtrará automáticamente.

Información Importante para Presupuestos

Importante: Los datos del cliente se usan para:
  • Encabezado de presupuestos
  • Datos fiscales en facturas
  • Encabezado de PDF
  • Aplicación de listas de precios específicas

Eliminar Cliente

Solo puede eliminar clientes que NO tengan presupuestos asociados.

  1. Seleccione el cliente
  2. Haga clic en Eliminar
  3. Confirme la eliminación en el diálogo
Alternativa: Si necesita mantener el registro pero no usarlo, marque el cliente como "Inactivo" en lugar de eliminarlo.

Listas de Precios por Cliente

Puede asignar listas de precios específicas a cada cliente:

  1. Abra el cliente a editar
  2. En la pestaña Precios, seleccione la lista de precios
  3. El sistema usará esta lista al crear presupuestos para este cliente

Datos Fiscales del Cliente

La información fiscal es crítica para la emisión de facturas:

Dato Fiscal Uso
CUIT Identificación en factura legal
Condición IVA Determina tipo de factura (A, B, C) a usar
Dirección y Localidad Domicilio del comprador en factura
Verificación: Antes de emitir una factura, verifique que los datos fiscales del cliente sean correctos en el sistema.

Mejores Prácticas

  • Código único: Use códigos que faciliten identificación (ej: números secuenciales)
  • Razón social exacta: Coincida con la registrada en AFIP
  • CUIT correcto: Verifique que sea correcto (evita problemas fiscales)
  • Email actualizado: Para enviar presupuestos y facturas
  • Condición IVA correcta: Es esencial para el tipo de factura
  • Datos completos: Complete todos los campos posibles
  • Revisión periódica: Actualice datos de clientes activos

Importación de Clientes

Si posee una lista de clientes en Excel o CSV, contacte al equipo técnico para una importación masiva.
