Guía de Inicio Rápido

Aprenda los conceptos básicos del Sistema Presupuestario en 10 minutos.

Paso 1: Ingresar al Sistema

Acceso: Al iniciar la aplicación, verá la pantalla de login
  1. Ingrese su usuario asignado
  2. Ingrese su contraseña (debe cambiarla en el primer ingreso)
  3. Haga clic en el botón Aceptar
Nota Importante: Por razones de seguridad, debe cambiar su contraseña en el primer acceso al sistema.

Paso 2: Entender la Ventana Principal

Una vez autenticado, verá la ventana principal del sistema con el siguiente diseño:

Componentes de la Interfaz:

  • Barra de Menú: Acceso a todas las funcionalidades del sistema
  • Área de Trabajo: Zona central donde se abren los formularios
  • Barra de Estado: Información del usuario conectado

Paso 3: Conceptos Básicos

Maestros de Datos

Son los datos base del sistema que debe mantener actualizado:

Elemento Descripción Acceso
Clientes Base de datos de sus clientes Menú > Maestros > Clientes
Productos Catálogo de productos/servicios Menú > Maestros > Productos
Vendedores Personal de ventas Menú > Maestros > Vendedores
Listas de Precio Precios por cliente o segmento Menú > Maestros > Listas de Precio

Flujo de Cotización a Factura

  1. Crear Presupuesto: Cotice productos a un cliente específico
  2. Gestionar Presupuesto: Modifique, apruebe o rechace
  3. Emitir Factura: Genere la factura legal del presupuesto aprobado
  4. PDF: El sistema genera automáticamente el PDF para enviar

Paso 4: Primeras Tareas

Para comenzar a usar el sistema, realice estos pasos en este orden:

1. Actualizar Configuración
Vaya a Menú > Configuración e ingrese los datos de su empresa (razón social, CUIT, dirección, etc.)
2. Crear Clientes
Ingrese sus clientes en Menú > Maestros > Clientes
3. Cargar Productos
Agregue sus productos en Menú > Maestros > Productos
4. Definir Precios
Cree listas de precios en Menú > Maestros > Listas de Precio
5. Crear Presupuesto
Haga su primera cotización en Menú > Cotizaciones > Gestionar Cotizaciones

Paso 5: Información Técnica

Consejo: El sistema utiliza una base de datos SQL Server para almacenar toda la información. Asegúrese de realizar copias de seguridad regularmente.

Requisitos del Sistema:

  • .NET Framework 4.8 o superior
  • SQL Server 2012 o superior
  • Windows 7 SP1 o superior
  • Mínimo 100 MB de espacio libre
