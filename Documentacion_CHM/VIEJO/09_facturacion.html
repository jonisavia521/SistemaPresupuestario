Facturación y Emisión de Facturas

Emita facturas legales a partir de presupuestos aprobados. Este es un proceso crítico que debe cumplir requisitos fiscales.

Conceptos Importantes

Factura: Documento legal emitido por AFIP que registra una venta. Tiene numeración secuencial y validez fiscal.
CAE: Código de Autorización Electrónico. Número único otorgado por AFIP para cada factura. Es obligatorio y válido por 60 días.
Punto de Venta: Identificador de la sucursal que emite. Formato: A00001-00000001 (PV + Número)

Acceso al Módulo de Facturación

Ruta: Menú Principal -> Venta -> Facturación

Aquí puede emitir facturas, consultar historial y generar documentos PDF.

Requisitos Previos para Facturar

Antes de emitir, asegúrese de:
  1. Presupuesto aprobado por el cliente
  2. Datos fiscales del cliente correctos (CUIT, Razón Social)
  3. Configuración de empresa completa
  4. Punto de venta configurado en Configuración
  5. Tipo de factura correcto (A, B, C)
  6. Conexión a base de datos disponible
  7. AFIP accesible (para obtener CAE)

Tipos de Facturas

Tipo Descripción Cliente IVA Formato
A Factura Tipo A (completa) Responsable Inscripto A00001-00000001
B Factura Tipo B Monotributista, Consumidor Final B00001-00000001
C Factura Tipo C Consumidor Final sin datos C00001-00000001

Emitir una Factura

Paso 1: Seleccionar Presupuesto

  1. En el módulo de Facturación, haga clic en Nueva Factura
  2. O abra un presupuesto aprobado y haga clic en Facturar
  3. El sistema cargará automáticamente los datos del presupuesto

Paso 2: Revisar Datos

Verifique que todos los datos sean correctos:

  • Empresa: Razón social, CUIT, dirección
  • Cliente: Nombre, CUIT, dirección, localidad
  • Detalles: Productos, cantidades, precios, IVA
  • Totales: Subtotal, IVA, ARBA, total final
  • Tipo de Factura: A, B o C (según cliente)

Paso 3: Emitir Factura

  1. Haga clic en Emitir Factura
  2. El sistema se conecta a AFIP para obtener el CAE
  3. Si es exitosa:
    • Se asigna número de factura secuencial
    • Se obtiene CAE válido por 60 días
    • Se registra en base de datos
    • Estado del presupuesto cambia a Facturado
  4. Si hay error, revise datos y reintente

Documento PDF

El sistema genera automáticamente un PDF profesional que incluye:

  • Logo de empresa (si está disponible)
  • Datos fiscales de empresa
  • Número de factura
  • CAE y código de barras
  • Información del cliente
  • Detalles de productos/servicios
  • Cálculos de impuestos
  • Total a pagar
  • Código QR (opcional)
  • Vencimiento del CAE

Descargar y Enviar Factura

  1. Abra la factura emitida
  2. Haga clic en Descargar PDF
  3. Se genera el documento en su carpeta de descargas
  4. Envíe por email al cliente

Historial de Facturas

Consulte todas las facturas emitidas:

  • Búsqueda: Por número, cliente, período
  • Filtros: Por estado, vendedor, rango de fechas
  • Acciones: Ver, descargar PDF, reenviar

ARBA - Percepción IIBB

En Buenos Aires, se retiene un porcentaje de la factura para ARBA (Ingresos Brutos):

Características:

  • Aplica: A empresas en AMBA (Área Metropolitana de Buenos Aires)
  • Porcentaje: Varía según tipo de cliente (típicamente 1-2%)
  • Cálculo: Se resta del total a pagar
  • Actualización: Se actualiza periódicamente

El sistema calcula automáticamente ARBA si está configurado según la provincia del cliente.

Errores Comunes en Facturación

Problema: "Error al conectar con AFIP"

Solución: Verifique que AFIP esté disponible y su conexión a Internet.

Problema: "Datos fiscales inválidos"

Solución: Verifique CUIT, razón social y datos de empresa en Configuración.

Problema: "Tipo de factura no válido para cliente"

Solución: Revise condición IVA del cliente. Debe coincidir con tipo de factura.

Problema: "Punto de venta no configurado"

Solución: Vaya a Configuración y complete datos de punto de venta.

Factura Anulada

Si necesita anular una factura emitida:

  1. No puede eliminarla de la base de datos (auditoría)
  2. Emita una "Nota de Crédito" (factura negativa)
  3. Documente el motivo de la anulación
  4. Registre en libros de AFIP

Contacte al equipo técnico para anulaciones de facturas.

Reportes Fiscales

Para auditoría y declaraciones:

  • Registro de Ventas: Por período
  • Total de IVA: Para DDJJ
  • ARBA Retenida: Para comprobación
  • Por Cliente: Análisis de ventas
  • Por Vendedor: Desempeño de equipo

Mejores Prácticas

  • Revisión completa: Antes de emitir, verifique todos los datos
  • Datos fiscales correctos: Evita rechazos de AFIP
  • Secuencial ordenado: No salte números de factura
  • Archivado: Guarde copias PDF para auditoría
  • Registro de cambios: Documente anulaciones
  • Respaldos: Backup regular de facturas
  • Cumplimiento fiscal: Declárelas a AFIP en tiempo y forma
