Solución de Problemas (Troubleshooting)

Encuentra soluciones a los problemas más comunes que podrías enfrentar.

Problemas de Acceso

No puedo iniciar sesión (usuario/contraseña incorrectos)
  1. Verifica que escribes correctamente tu usuario
  2. Confirma que tu contraseña es correcta (mayúsculas/minúsculas)
  3. Si olvidaste contraseña, contacta al administrador
  4. Intenta desactivar Bloq Mayús si está activado
Aplicación se abre pero no puedo ver datos
  1. Verifica que tengas permisos para ese módulo
  2. Contacta al administrador para asignar permisos
  3. Algunos módulos solo están disponibles para ciertos roles

Problemas de Conexión a BD

Base de datos no disponible / No puedo conectar
  1. Verifica que tu PC esté conectada a la red
  2. Pincha ping al servidor SQL: cmd ? ping [nombre_servidor]
  3. Verifica que SQL Server esté corriendo (contacta a IT)
  4. Reinicia la aplicación y/o tu computadora
  5. Si persiste, contacta a soporte técnico
La aplicación está muy lenta
  1. Verifica tu conexión de red (ancho de banda disponible)
  2. Cierra otras aplicaciones que consuman recursos
  3. Si es servidor SQL remoto, contacta a IT (puede estar saturado)
  4. Prueba reiniciando la aplicación
  5. Si sigue lenta, contacta a soporte

Problemas con Presupuestos

No puedo crear presupuesto - error "Cliente no válido"
  1. Asegúrate de seleccionar un cliente activo
  2. Si el cliente está desactivado, no aparecerá en la lista
  3. Ve a Maestros > Clientes y reactiva el cliente si es necesario
  4. Intenta nuevamente
Presupuesto no se guarda - error en validación
  1. Verifica que todos los campos obligatorios estén completos
  2. Revisa mensajes de error (indican qué campo tiene problema)
  3. Asegúrate de haber seleccionado al menos un producto
  4. Valida que los números sean correctos (cantidades, precios)

Problemas con Facturación

No puedo emitir factura - "Datos fiscales inválidos"
  1. Ve a Archivo > Configuracion, revisa datos de empresa
  2. Verifica CUIT, Razón Social, Dirección (deben estar completos)
  3. Abre cliente y revisa su CUIT y Tipo IVA
  4. Asegúrate que Tipo IVA del cliente coincida con tipo de factura
  5. Intenta nuevamente
Error al obtener CAE - "No hay conexión a AFIP"
  1. Verifica que tengas conexión a Internet
  2. Abre navegador y accede a www.afip.gov.ar para probar conectividad
  3. Revisa estado de AFIP (a veces está en mantenimiento)
  4. Intenta facturar más tarde
  5. Si persiste, contacta a soporte
AFIP rechazó factura - error en los datos
  1. Lee el mensaje de error de AFIP con cuidado
  2. Los errores típicos son: CUIT inválido, Razón Social incorrecta, Tipo IVA incoherente
  3. Contacta a tu contador o AFIP directamente para aclarar
  4. Corrige los datos y reintenta

Problemas de Datos

Cambié un cliente pero los datos antiguos siguen apareciendo
  1. Los datos en presupuestos/facturas se guardan en el momento de creación
  2. Los cambios futuros solo afectarán nuevos presupuestos
  3. Esto es intencional (auditoría fiscal)
  4. Es normal que veas datos inconsistentes entre presupuestos antiguos y nuevos
Presupuesto desapareció - ¿dónde está?
  1. Verifica el filtro de búsqueda (no está filtrado por estado)
  2. Desactiva filtro "Solo ver activos" si aplica
  3. Busca por número o cliente en la barra de búsqueda
  4. Revisa fecha de emisión (filtro por período)
  5. Presupuestos borrados siguen en la BD pero con estado "Borrado"

Problemas de Backup/Restauración

El backup tarda mucho en crearse
  1. Es normal si tienes muchos datos acumulados
  2. No cierres la aplicación durante el proceso
  3. Espera a que termine (puede tardar minutos)
  4. Usa esta información para programar backups en horas de baja actividad
No puedo restaurar - error en archivo
  1. Verifica que el archivo de backup existe y es accesible
  2. Confirma que no esté en carpeta de solo lectura
  3. Prueba con otro backup (quizás ese está corrupto)
  4. Si ninguno funciona, contacta a soporte

Problemas de Idioma

El sistema está en idioma incorrecto
  1. Ve a Archivo > Configuracion > Selección de Idioma
  2. Selecciona el idioma que prefieres
  3. Haz clic en "Cambiar Idioma"
  4. El cambio es inmediato

Pasos Generales de Troubleshooting

  1. Relee el mensaje de error: A menudo contiene la solución
  2. Google el error: Otros pueden haber tenido el mismo problema
  3. Reinicia: Cierra y abre la aplicación nuevamente
  4. Borrador caché: A veces hay datos en caché que causan conflictos
  5. Verifica conectividad: Red, Base de datos, Internet
  6. Contacta soporte: Si nada funciona, pide ayuda profesional

Cuándo Contactar a Soporte

  • Errores que persisten después de reiniciar
  • Problemas de conexión a base de datos
  • Facturación rechazada por AFIP
  • Datos faltantes o corruptos
  • Comportamientos extraños sin explicación
