Facturación

Emite facturas legales a partir de presupuestos aprobados. Las facturas son documentos legales ante AFIP con obligaciones tributarias.

Acceso

Ruta: Ventas > Facturación

Requisitos para Facturar

Obligatorio tener ANTES de facturar:

  • Presupuesto en estado "Aprobado"
  • Datos correctos de cliente (CUIT, Razón Social, Tipo IVA)
  • Configuración de empresa completa (Razón Social, CUIT, Dirección)
  • Punto de venta configurado en Configuracion
  • Conexión a Internet disponible (para obtener CAE de AFIP)

Proceso de Facturación

Paso 1: Seleccionar Presupuesto(s)

  1. En Ventas > Facturación, verás lista de presupuestos aprobados
  2. Selecciona uno o varios presupuestos (puedes facturar masivo)
  3. Sistema muestra cliente, cantidad de presupuestos, total

Paso 2: Emitir Factura

  1. Revisa datos: empresa, cliente, detalles, totales
  2. Verifica que todo sea correcto ANTES de emitir
  3. Haz clic en Emitir Factura o Obtener CAE
  4. Sistema se conecta con AFIP para obtener CAE

Paso 3: Guardar y Descargar PDF

  1. Si todo es correcto, sistema asigna:
    • Número de factura secuencial (ej: A00001-00000001)
    • CAE de AFIP (14 dígitos)
    • Fecha de vencimiento CAE (fecha + 60 días)
  2. Presupuesto cambia automáticamente a estado "Facturado"
  3. Sistema pregunta si deseas generar PDF
  4. Descarga y guarda el PDF para enviar/archivar

Información en Factura

Sección Qué Incluye
Encabezado Número, tipo, CAE, fecha, vencimiento CAE
Datos Empresa Razón Social, CUIT, Dirección, Localidad
Datos Cliente Razón Social, CUIT, Condición IVA
Detalles Productos, cantidades, precios, descuentos
Totales Subtotal, IVA, ARBA, Total Final

Facturación Masiva

Puedes facturar varios presupuestos del MISMO cliente simultáneamente:

  1. Selecciona múltiples presupuestos (todos del mismo cliente)
  2. El sistema genera una factura con todos los detalles combinados
  3. Se emite UNA factura con múltiples presupuestos
  4. Útil para clientes con muchas órdenes a la vez

Errores Comunes

Error: "Datos fiscales inválidos"

Causa: Cliente o empresa con datos incompletos o incorrectos.

Solución: Revisa Configuracion (empresa) y cliente. Asegúrate CUIT, Razón Social y Tipo IVA sean correctos.

Error: "No hay conexión a AFIP"

Causa: Problema de conectividad o AFIP no disponible.

Solución: Verifica Internet. Intenta más tarde. Revisa status.afip.gov.ar.

Error: "Tipo de factura no válido para cliente"

Causa: Tipo IVA del cliente no coincide con tipo de factura configurado.

Solución: Verifica tipo IVA del cliente. Debe coincidir con tu punto de venta (A, B o C).

CAE - Lo que Debes Saber

  • ¿Qué es? Código único de AFIP que autoriza la factura
  • ¿Cuándo lo obtengo? Al emitir factura (descarga automática)
  • ¿Vigencia? 60 días desde emisión
  • ¿Es obligatorio? Sí, sin CAE la factura no es válida
  • ¿Qué pasa si vence? Factura pierde validez. Debes emitir Nota de Crédito.

PDF de Factura

El sistema genera un PDF profesional que incluye:

  • Logo de empresa (si está disponible)
  • Número, tipo, CAE, fecha
  • Código de barras del CAE
  • Datos fiscales de empresa y cliente
  • Detalles completos de productos
  • Cálculos e impuestos desglosados
  • Total final a pagar
  • Fecha de vencimiento del CAE

Historial de Facturas

Puedes consultar todas las facturas emitidas:

  • Búsqueda por número, cliente, período
  • Filtros por estado, vendedor, rango de fechas
  • Ver detalles, descargar PDF, reenviar

ARBA en Facturas

Si el cliente está en Buenos Aires, la factura muestra:

  • Línea de "Percepción ARBA" con el porcentaje
  • Monto retenido para IIBB
  • Se resta del total a pagar al cliente

Mejores Prácticas

  • Revisa antes de facturar: Verifica todos los datos
  • Copia exacta de presupuesto: No debe haber discrepancias
  • Guarda PDF: Archivo local + copia en la nube
  • Verifica CAE: Asegúrate que aparezca en el PDF
  • Registro ordenado: Mantén facturas numeradas secuencialmente
  • Declaración a AFIP: Incluye todas las facturas en declaraciones juradas
