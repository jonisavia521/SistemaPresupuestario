Antes de Contactar Soporte

Cuando contactes a soporte, ten listo:

  • Descripción exacta del problema
  • Pasos para reproducir el error
  • Mensaje de error exacto (si hay)
  • Tu usuario y hora aproximada del problema
  • Información del sistema (versión del SO, navegador, etc.)

Contacto de Soporte

Tu organización debería proporcionarte:

  • Email de soporte técnico
  • Teléfono de soporte
  • Horario de atención
  • Nombre del responsable de IT

Problemas Comunes y Soluciones Rápidas

Aplicación no abre: Reinicia tu computadora, verifica conexión de red.

Base de datos no accesible: Verifica que SQL Server esté disponible. Contacta a soporte.

Mensajes en idioma incorrecto: Cambia idioma en Configuración.

Datos faltantes: Verifica que el backup esté actualizado.

Factura rechazada por AFIP: Revisa datos fiscales. Contacta a soporte.
