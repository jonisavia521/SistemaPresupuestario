Configuración del Sistema

La configuración correcta del sistema es fundamental. Aquí definirás los datos de tu empresa que aparecerán en todas tus facturas y documentos. Es el punto de partida para trabajar eficientemente.

Acceso a Configuración

Ruta: Menú Principal ? Archivo ? Configuración

Se abrirá la ventana de Configuracion con 3 pestañas principales.

Pestaña 1: Configuración General de la Empresa

Datos que Debes Ingresar

Campo Descripción Requerido Nota
Razón Social Nombre oficial de tu empresa registrado ante AFIP SÍ Máx 255 caracteres. Debe coincidir con AFIP.
CUIT Código Único de Identificación Tributaria (11 dígitos) SÍ Ingresa solo números. Sistema valida formato.
Tipo de IVA Tu condición fiscal ante AFIP SÍ Responsable Inscripto, Monotributista, etc.
Dirección Domicilio fiscal de tu empresa SÍ Calle y número. Aparece en facturas.

Teléfono Tu número de contacto principal NO Incluye código de área. Ej: (011) 1234-5678

Cómo Completar Configuración General

  1. Ve a Archivo ? Configuración
  2. Asegúrate de estar en la solapa "Configuración General"
  3. Completa todos los campos marcados como requeridos
  4. Verifica que los datos coincidan con tu inscripción en AFIP
  5. Haz clic en Guardar Configuración

Consejo: Antes de guardar, descarga tu información de AFIP por internet y cópiala exactamente. Errores de tipeo causan problemas en facturas.

Pestaña 2: Selección de Idioma

Una de las características más útiles del sistema es el cambio de idioma completamente dinámico. No necesitas reiniciar la aplicación.

Idiomas Disponibles

Idioma Código Descripción
Español es-AR Español (Argentina) - El idioma por defecto
English en-US Inglés (Estados Unidos) - Para usuarios internacionales

Cómo Cambiar el Idioma

  1. En la ventana de Configuración, ve a la solapa "Selección de Idioma"
  2. Veras dos botones: Español o English
  3. Selecciona el idioma que prefieres
  4. Haz clic en Cambiar Idioma
  5. El sistema cambiará todos los textos inmediatamente

Lo que sucede automáticamente cuando cambias idioma:

  • Todos los formularios abiertos se actualizan
  • Etiquetas, botones y mensajes cambian de idioma
  • La configuración se guarda automáticamente
  • No necesitas reiniciar la aplicación

Nota importante: Los datos (clientes, presupuestos, etc.) NO se traducen. Solo la interfaz del sistema cambia de idioma. Tus datos siguen siendo en español.

Pestaña 3: Backup y Restauración

El backup es CRÍTICO para proteger tus datos. Una copia de seguridad es tu red de contención ante pérdida de información.

¿Por Qué Hacer Backups?

  • Protección contra fallos: Tu computadora puede fallar sin previo aviso
  • Recuperación ante desastres: Un virus, corte de luz, etc.
  • Cumplimiento fiscal: Debes tener respaldo de datos fiscales
  • Auditoría: Tener backups demuestra responsabilidad

Crear un Backup (Copia de Seguridad)

  1. Ve a Archivo ? Configuración ? Pestaña "Backup"
  2. Haz clic en el botón "Crear Backup"
  3. Se abrirá un diálogo para elegir dónde guardar el archivo
  4. Se sugiere un nombre automático con la fecha (ej: SistemaPresupuestario_Backup_20240115_143045.bak)
  5. Elige una carpeta segura (externa, nube, unidad USB)
  6. Haz clic en Guardar
  7. El sistema creará el backup (puede tardar unos segundos)
  8. Verás un mensaje de éxito

Historial de Backups

El sistema mantiene un historial de todos los backups que has creado. En la grilla "Historial de Backups" verás:

  • Fecha y hora de creación
  • Ruta donde está guardado
  • Tamaño del archivo
  • Usuario que lo creó

Restaurar desde un Backup

  1. En la pestaña de Backup, selecciona el backup que deseas restaurar de la lista
  2. Haz clic en "Restaurar"
  3. ¡ADVERTENCIA CRÍTICA! Se mostrará un mensaje advirtiéndote que:
    • Se perderán TODOS los datos actuales
    • Se reemplazarán con los del backup
    • Los cambios después del backup se perderán
  4. Si estás seguro, confirma haciendo clic en SÍ
  5. El sistema restaurará los datos (puede tardar)
  6. La aplicación se reiniciará automaticamente
CUIDADO: La restauración de un backup reemplaza TODOS tus datos actuales. Úsala solo si sabes lo que haces y es NECESARIO. Una vez confirmada, no hay vuelta atrás.

Frecuencia Recomendada de Backups

Situación Frecuencia Recomendada Nota
Uso diario normal Una vez por semana Al final de cada semana de trabajo
Mucho movimiento (presupuestos/facturas) Cada dos días O al final de cada jornada importante
Cierre de período fiscal Antes y después Protege contra cambios accidentales
Antes de actualizaciones del sistema Obligatorio Así puedes revertir si hay problema

Mejores Prácticas para Backups

  • Guarda en múltiples lugares: Disco local + unidad externa + nube
  • Organiza tus backups: Crea carpetas por mes/año
  • Automatiza si es posible: Program un horario fijo
  • Verifica la integridad: Ocasionalmente, prueba restaurar un backup antiguo
  • Documentación: Anota cuándo hiciste cada backup
  • Rotación: Mantén 3-5 backups recientes, elimina los muy antiguos

Validación de Configuración

Antes de empezar a facturar, el sistema valida automáticamente que tengas configuración básica:

Si faltan datos, veras este mensaje: "No se encontró configuración del sistema. Por favor, complete primero los datos de configuración en la pestaña 'Configuración General'."

Significa que no puedes continuar hasta llenar los campos requeridos. Vuelve a Configuración y completa todo.

Guardado de Cambios

Todos los cambios en Configuración se guardan en la base de datos del sistema.

Acción Qué Sucede
Cambias datos y haces clic en "Guardar Configuración" Los datos se guardan inmediatamente en la BD
Cambias idioma y haces clic en "Cambiar Idioma" El idioma se guarda y la interfaz cambia al instante
Cierras la ventana sin guardar cambios Los cambios NO se guardan
Haces clic en "Crear Backup" o "Restaurar" Se ejecuta la operación en la BD

Problemas Comunes en Configuración

Problema: "CUIT inválido"

Causa: Ingresaste caracteres no numéricos o menos de 11 dígitos.

Solución: Ingresa solo los 11 números del CUIT, sin guiones ni espacios.

Problema: No aparecen provincias en el combobox

Causa: La conexión a la BD se perdió al cargar.

Solución: Cierra la ventana y vuelve a intentar. Si persiste, contacta a soporte.

Problema: Backup tarda mucho tiempo

Causa: La base de datos es muy grande o tu computadora es lenta.

Solución: Es normal. Espera a que termine. No cierres la aplicación.

Consejos Finales

  • Revisa tu configuración: Cada 6 meses, verifica que datos fiscales sigan siendo correctos
  • Sincroniza con AFIP: Si cambias datos en AFIP, actualízalos también aquí
  • Prueba el backup: Ocasionalmente, restaura uno antiguo para comprobar que funciona
  • Mantén historiales: No elimines backups muy antiguos sin razón
