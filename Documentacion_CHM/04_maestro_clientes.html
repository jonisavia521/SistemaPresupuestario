Gestión de Clientes

Los clientes son el corazón de cualquier negocio. En este módulo, administrarás toda la información de tus clientes: datos fiscales, condiciones de pago, vendedor asignado, lista de precios y más. Todos tus presupuestos y facturas se basarán en estos datos.

Acceso al Módulo

Ruta: Menú Principal ? Maestros ? Clientes

Se abrira la ventana de Gestión de Clientes con una lista de todos tus clientes registrados.

La Pantalla Principal de Clientes

Elementos Principales

  • Tabla de clientes: Muestra todos los clientes registrados con información básica
  • Búsqueda: Campo para buscar clientes por código, nombre o CUIT
  • Botones de acción: Nuevo, Editar, Desactivar/Reactivar, Cerrar
  • Filtro "Solo ver activos": Checkbox para mostrar solo clientes activos

Crear un Nuevo Cliente

Paso a Paso

  1. Haz clic en el botón "Nuevo"
  2. Se abrira el formulario de alta de cliente
  3. Completa todos los campos (ver tabla abajo)
  4. Haz clic en "Aceptar" para guardar

Campos del Formulario de Cliente

Campo Descripción Requerido Notas
Código Cliente Identificador único (asignado automáticamente) Sí Formato: CLI-yyyyMMddHHmmss. No editable.
Razón Social Nombre oficial del cliente (empresa o persona) Sí Tal como aparece en sus documentos fiscales
Tipo Documento DNI, CUIT o CUIL Sí CUIT es el más común para empresas
Número Documento Número del documento (solo dígitos) Sí Sistema valida solo números. CUIT = 11 dígitos.
Tipo IVA Condición fiscal del cliente ante AFIP Sí Responsable Inscripto, Monotributista, Consumidor Final, etc.
Condición de Pago Plazo de pago (Contado, 30, 60, 90, 120 días) Sí Para presupuestos y facturas
Alícuota ARBA Porcentaje de percepción IIBB (si aplica) No Entre 0 y 100. Solo si está en Buenos Aires.
Provincia Provincia donde opera el cliente No Afecta cálculos de impuestos. Puede ser "Sin provincia".
Vendedor Asignado Vendedor responsable de este cliente No Presiona F3 o Enter para abrir selector. Opcional.
Dirección Domicilio del cliente No Calle, número y detalles. Aparece en facturas.
Localidad Ciudad del cliente No Ej: Buenos Aires, La Plata, Córdoba
Teléfono Número de contacto No Incluye código de área. Ej: (011) 1234-5678
Email Correo electrónico del cliente No Para enviar presupuestos y facturas

Información Importante

CRÍTICO para Facturación: Los datos de Razón Social, CUIT, Tipo IVA y Condición Pago deben ser exactos. Errores aquí causan problemas en facturas y rechazos de AFIP.

Editar un Cliente

  1. En la lista de clientes, selecciona el cliente que deseas editar
  2. Haz clic en el botón "Editar"
  3. Se abrira el formulario con los datos del cliente
  4. Modifica los campos que necesites
  5. Haz clic en "Aceptar" para guardar los cambios

Nota: El Código Cliente no se puede editar. Si necesitas cambiar un cliente completo, es mejor crear uno nuevo.

Desactivar un Cliente

Si un cliente ya no opera contigo pero quieres mantener el historial (presupuestos, facturas), desactívalo en lugar de eliminarlo.

  1. Selecciona el cliente en la lista
  2. Haz clic en "Desactivar"
  3. Confirma en el diálogo emergente
  4. El cliente se marca como inactivo

Qué Sucede Cuando Desactivas un Cliente

  • No aparecerá en los selectores de clientes para nuevos presupuestos
  • Puedes reactivarlo en cualquier momento si es necesario
  • El historial de presupuestos y facturas se mantiene intacto
  • Es mejor que eliminar porque conserva la auditoría fiscal

Reactivar un Cliente

  1. Asegúrate de tener el filtro "Solo ver activos" DESACTIVADO
  2. Selecciona el cliente inactivo que deseas reactivar
  3. Haz clic en "Reactivar"
  4. El cliente vuelve a estado activo

Buscar un Cliente

Si tienes muchos clientes, usa la búsqueda para encontrar rápidamente.

  1. En el campo de búsqueda, ingresa el código, nombre o CUIT del cliente
  2. El sistema filtrará automaticamente mientras escribes
  3. Los resultados se muestran en la tabla
  4. Para limpiar la búsqueda, borra el texto

Ejemplos de Búsqueda

Qué Buscas Ingresa Resultado
Cliente por código CLI-202401 Muestra clientes con ese código
Cliente por nombre parcial Acme Muestra "Acme S.A.", "Acmex Ltd", etc.
Cliente por CUIT 20156789012 Muestra el cliente con ese CUIT

Asignar Vendedor a Cliente

Puedes asignar un vendedor específico a un cliente para seguimiento de comisiones.

  1. En el formulario de cliente, busca el campo "Código Vendedor"
  2. Presiona F3 o Enter para abrir el selector de vendedores
  3. Se abrira una ventana con la lista de vendedores activos
  4. Busca y selecciona el vendedor que deseas
  5. Haz clic en "Seleccionar"

Asignar Lista de Precios

Puedes asignar una lista de precios específica a un cliente para ofertas personalizadas.

  • Si el cliente tiene lista de precios asignada, esa se usará en nuevos presupuestos
  • Si no tiene, se usarán los precios base de los productos
  • Puedes cambiar los precios dentro de cada presupuesto manualmente si es necesario

Datos Fiscales - Información Crítica

Estos datos DEBEN ser exactos para facturación legal:

  • Razón Social: Copiar exactamente del certificado de AFIP
  • CUIT: Validar contra AFIP antes de guardar
  • Tipo IVA: Consultar con el cliente o AFIP si no estás seguro
  • Condición Pago: Acordar con el cliente o establecer política

Padrón ARBA

Para clientes en Buenos Aires, el campo "Alícuota ARBA" es importante.

  • El porcentaje se actualizapor ARBA periódicamente
  • El sistema puede actualizar masivamente todos los clientes en "Venta ? Actualizar Padrón ARBA"
  • Se retiene automáticamente en facturas

Mejores Prácticas para Clientes

  • Verifica datos: Antes de guardar, revisa que todo sea correcto especialmente datos fiscales
  • Mantén actualizado: Si un cliente cambia de datos, actualízalos inmediatamente
  • Desactiva en lugar de eliminar: Conserva historial fiscal
  • Una vez guardado: Los datos en presupuestos y facturas no se modifican automáticamente
  • Cuidado con duplicados: Verifica no crear el mismo cliente dos veces
  • Estandariza nombres: Usa nombres consistentes para el mismo cliente (ej: siempre "Acme S.A." no "ACME" o "acme")

Problemas Comunes

Problema: "No puedo encontrar el cliente en el selector de presupuestos"

Causa: El cliente está desactivado.

Solución: Reactívalo o crea un cliente nuevo.

Problema: "Sistema rechaza el CUIT como inválido"

Causa: Ingresaste caracteres no numéricos.

Solución: Ingresa solo los 11 números del CUIT, sin guiones ni espacios.

Problema: "Cambié datos del cliente pero no aparecen en presupuestos anteriores"

Causa: Los datos en presupuestos se guardan en el momento de creación.

Solución: Es así por diseño. Los cambios futuros solo afectarán nuevos presupuestos.
