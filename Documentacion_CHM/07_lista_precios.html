Gestión de Listas de Precios

Crea y gestiona listas de precios personalizadas para ofrecerle a diferentes clientes o segmentos precios diferenciados.

Concepto

Una lista de precios es un conjunto de precios específicos que se aplica a un cliente o grupo de clientes. Permite tener precios mayoristas, ofertas especiales, etc.

Crear Lista de Precios

Ve a Maestros > Listas de Precios > Nueva. Asigna código y nombre único, agrega productos con sus precios personalizados, guarda.

Asignar a Cliente

Una vez creada la lista, abre un cliente en Maestros > Clientes > Editar, selecciona la lista en el campo correspondiente, guarda.

Uso en Presupuestos

Cuando creas un presupuesto para un cliente con lista asignada, los precios se cargan automáticamente de esa lista. Puedes modificarlos dentro del presupuesto si es necesario.

Desactivar Lista

Si una lista ya no se usa, desactívala. Los presupuestos existentes mantienen la lista, pero no aparecerá en nuevas asignaciones.

Ejemplo Práctico

Escenario: Ofreces descuentos a mayoristas. Creas una "Lista Mayorista" con precios reducidos, la asignas a clientes mayoristas, y automáticamente todos sus presupuestos usan esos precios.
