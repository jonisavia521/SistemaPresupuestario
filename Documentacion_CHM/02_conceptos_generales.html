Conceptos Generales del Sistema Presupuestario

Conoce la terminología y conceptos basicos que necesitas para entender como funciona el sistema. Esta información es importante para trabajar eficientemente.

Términos Clave - Debes Conocerlos

Presupuesto (o Cotización): Es un documento comercial donde detallas los productos o servicios que ofreces a un cliente, con sus cantidades, precios, condiciones y vigencia. NO es un documento legal en sí mismo, sino una propuesta comercial. Es la base para emitir facturas.
Cotización: Sinónimo de presupuesto. El término se usa indistintamente. "Gestionar Cotizaciones" = "Gestionar Presupuestos".
Factura: Documento legal y obligatorio emitido por AFIP. Registra una venta real y debe tener numeración secuencial. Solo se puede emitir de un presupuesto que esta APROBADO.
CAE (Código de Autorización Electrónico): Número único de 14 dígitos que otorga AFIP a cada factura emitida. Es obligatorio, prueba que la factura fue registrada ante el organismo y tiene una vigencia de 60 días.
ARBA (Agencia de Recaudación de Impuestos): Organismo del Estado de Buenos Aires que recauda el impuesto de Ingresos Brutos (IIBB). El sistema puede retener un porcentaje de las facturas para esta agencia automáticamente.
Punto de Venta (PV): Identificador de tu sucursal que emite facturas. Tiene formato: A00001-00000001 o B00001-00000001. Debes configurarlo en Configuracion para poder facturar.
IVA (Impuesto al Valor Agregado): Impuesto nacional sobre las ventas. El más común es 21% (General), también hay 10.5% (Reducido) y algunos productos están exentos. Se calcula automaticamente en el sistema.
Lista de Precios: Conjunto de precios especificos para un cliente o grupo de clientes. Permite ofrecer precios diferenciados sin cambiar el precio base del producto.

Tipos de Clientes Según Situación Fiscal

Tipo de IVA Descripción Factura Emitida Nota
Responsable Inscripto Empresa obligada a facturar. Paga IVA. Tipo A (completa) La más común. Puede aplicar ARBA.
Monotributista Pequeño empresa o autonomo. Sistema simplificado. Tipo B No discrimina IVA en factura.

Estados de un Presupuesto - El Ciclo de Vida

1. Emitido

Es el estado inicial cuando creas un presupuesto. El documento existe en el sistema pero aún no ha sido aprobado ni facturado. Puedes:

  • Editar los datos
  • Agregar o eliminar productos
  • Cambiar precios y descuentos
  • Aprobar si el cliente aceptó
  • Rechazar si el cliente no quiso
  • Copiar para crear similar
  • Generar PDF para enviar

2. Aprobado

El cliente aprobó la propuesta. Ahora esta listo para facturar. Puedes:

  • Emitir factura legal
  • Aún puedes editar si es necesario
  • Copiar para crear otro presupuesto similar
  • Ver el PDF

3. Rechazado

El cliente no aceptó la propuesta. El presupuesto queda como referencia histórica pero no puede facturarse. Puedes:

  • Consultarlo si es necesario
  • Copiarlo para crear una nueva propuesta mejorada
  • Eliminarla si deseas

4. Facturado

Se emitió una factura legal de este presupuesto. Es un documento definitivo. Puedes:

  • Verla y descargar el PDF
  • Consultarla si es necesario (auditoría)
  • Copiarla para crear un nuevo presupuesto (no copia la factura)
Importante: Un presupuesto SOLO puede pasar a estado Facturado si esta APROBADO. No puedes saltear ese paso.

Cálculos Automáticos - Como Funciona la Matemática

Cálculo de un Detalle (una línea de presupuesto):

Subtotal = Cantidad × Precio Unitario Descuento = Subtotal × (Descuento% / 100) Base IVA = Subtotal - Descuento IVA = Base IVA × (% IVA / 100) Total Línea = Base IVA + IVA

Ejemplo Práctico:

Concepto Cálculo Monto
Producto: Laptop Premium -
Cantidad: 2 unidades -
Precio Unitario: - $1.000
Subtotal: 2 × $1.000 $2.000
Descuento 10%: $2.000 × 10% -$200
Base IVA: $2.000 - $200 $1.800
IVA 21%: $1.800 × 21% $378
Total Línea: $1.800 + $378 $2.178

Totales del Presupuesto Completo:

Subtotal General = Suma de todos los Subtotales Total Descuentos = Suma de todos los Descuentos Total IVA = Suma de todos los IVA ARBA (si aplica) = Subtotal × % ARBA TOTAL FINAL = Subtotal - Descuentos + IVA - ARBA

Configuración por Provincia - ARBA

ARBA solo aplica en Buenos Aires. El sistema detecta la provincia del cliente y calcula automáticamente la percepción si corresponde.

Los porcentajes varían según la actividad. En el sistema, puedes ver y actualizar estos valores en Configuracion.

Seguridad y Permisos - Quién Puede Hacer Qué

El sistema tiene un sistema de permisos basado en Familias y Patentes. Según tu rol de usuario, tendrás acceso a diferentes módulos:

Rol Puede Ver Puede Hacer
Administrador Todo Crear, editar, eliminar, gestionar usuarios
Vendedor Clientes, productos, presupuestos Crear y editar presupuestos
Aprobador Presupuestos emitidos Aprobar o rechazar presupuestos
Facturador Presupuestos aprobados Emitir facturas

Flujo de Trabajo Típico - De Principio a Fin

  1. Vendedor crea presupuesto: Selecciona cliente, agrega productos
  2. Presupuesto en estado "Emitido": Se envía al cliente
  3. Aprobador revisa: Verifica los datos sean correctos
  4. Aprobador aprueba: Presupuesto pasa a estado "Aprobado"
  5. Facturador emite factura: De presupuesto aprobado a factura legal
  6. Sistema obtiene CAE: Se conecta con AFIP y obtiene número
  7. Factura lista: Se genera PDF para enviar al cliente
  8. Presupuesto en estado "Facturado": Queda como referencia histórica
