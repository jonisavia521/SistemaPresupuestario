Gestión de Presupuestos y Cotizaciones

Este es el módulo central del sistema. Aquí crearás, editarás, aprobarás y gestionarás todos tus presupuestos comerciales antes de facturar.

Acceso al Módulo

Ruta: Cotizaciones > Gestionar Cotizaciones (para crear/editar) o Cotizaciones > Aprobar Cotizaciones (para revisar aprobaciones)

Crear un Nuevo Presupuesto

Paso 1: Encabezado

  1. Haz clic en Nuevo
  2. Selecciona cliente (requerido) - presiona F3 o Enter para buscar
  3. Fecha se completa automáticamente con hoy
  4. Ingresa vigencia (días que el presupuesto es válido)
  5. Opcional: Asigna vendedor, agrega observaciones

Paso 2: Agregar Productos

  1. En la sección "Detalles", haz clic en Nuevo
  2. Selecciona producto (presiona F3 o Enter)
  3. Ingresa cantidad
  4. El precio se carga automáticamente (puedes editarlo)
  5. Opcional: ingresa descuento %
  6. Sistema calcula automáticamente subtotal, IVA y total
  7. Haz clic en Guardar
  8. Repite para cada producto

Paso 3: Revisar y Guardar

  1. Verifica todos los datos: cliente, productos, precios, totales
  2. Haz clic en Guardar para crear el presupuesto
  3. El sistema asigna número único automáticamente
  4. El presupuesto queda en estado "Emitido"

Estados del Presupuesto

Estado Descripción Puedes...
Emitido Creado, en espera de aprobación Editar, Aprobar, Rechazar, Copiar, Eliminar, Generar PDF
Aprobado Cliente aprobó - listo para facturar Editar, Facturar, Copiar, Ver, Generar PDF
Rechazado Cliente no aceptó Ver, Copiar como nuevo, Eliminar
Facturado Se emitió factura legal Ver, Descargar PDF, Copiar
Borrado Eliminado lógicamente Ver (solo consulta)

Editar un Presupuesto

  1. Selecciona el presupuesto en la lista
  2. Haz clic en Editar
  3. Modifica los datos que necesites
  4. Haz clic en Guardar

Nota: Solo puedes editar presupuestos en estado "Emitido" o "Aprobado".

Aprobar un Presupuesto

  1. Selecciona presupuesto en estado "Emitido"
  2. Haz clic en Aprobar
  3. Confirma la acción
  4. El estado cambia a "Aprobado"

¡Crítico! Solo presupuestos aprobados pueden facturarse.

Rechazar un Presupuesto

  1. Selecciona presupuesto en estado "Emitido"
  2. Haz clic en Rechazar
  3. El estado cambia a "Rechazado"

Copiar un Presupuesto

Útil para crear presupuestos similares sin volver a cargar todos los datos:

  1. Selecciona el presupuesto a copiar
  2. Haz clic en Copiar
  3. Se crea uno nuevo con los mismos datos
  4. Modifica lo que necesites y guarda

Generar PDF

  1. Abre el presupuesto que deseas imprimir
  2. Haz clic en Imprimir o Generar PDF
  3. El sistema crea un documento PDF profesional
  4. Incluye tus datos de empresa, cliente, detalles, cálculos e impuestos
  5. Puedes guardarlo en tu computadora para enviar o archivar

Búsqueda y Filtros

Filtra presupuestos por:

  • Cliente (nombre o código)
  • Estado (Emitido, Aprobado, Rechazado, Facturado)
  • Período (fecha desde-hasta)
  • Vendedor
  • Número de presupuesto

Cálculos Automáticos

El sistema calcula automáticamente:

  • Subtotal: Suma de cantidad × precio (antes de IVA)
  • Descuentos: Aplicados a cada línea
  • IVA: Según % configurado en cada producto
  • ARBA: Si el cliente está en Buenos Aires
  • Total Final: Subtotal - Descuentos + IVA - ARBA

Antes de Facturar

Requisitos:

  • Presupuesto DEBE estar en estado "Aprobado"
  • Todos los datos del cliente deben ser correctos
  • Configuración de empresa debe estar completa
  • Punto de venta debe estar configurado
  • Debes tener acceso a AFIP para obtener CAE

Mejores Prácticas

  • Revisa antes de guardar: Verifica cliente, productos, precios, totales
  • Datos completos: No dejes campos importantes en blanco
  • Vigencia realista: Típicamente 15-30 días
  • Observaciones claras: Términos, pagos, entregas especiales
  • Genera PDF antes de enviar: Asegúrate de que se vea bien
  • Sigue el flujo: Emitido > Aprobado > Facturado
